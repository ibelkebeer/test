Naive Look through the board cells linearly, and if there is an empty cell, cycle through numbers 1 through 9, and when a number doesn't conflict with its clique, place that number in the cell and move on to the next cell. Backtrack when there is no non-conflicting guess for a cell
Smarter-1 Detirmine what values fit into the empty cells and fill them with those values starting with the cells that have the least number of possible values.
Smarter-2 Same as smarter-1, but when a guess is placed in a cell, remove that guess from all other cells in its clique. If this lowers the number of guesses for a cell to 1, remove that guess from all other cells in its clique and so on. If this lowers the number of guesses for a cell to 0, then the original guess for the first cell is invalid.
